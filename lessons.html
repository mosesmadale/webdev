Lets start!
We are going to start with HTML since it is the easiest and can help you output into the Browser
  • Step 1
You will need the right tools to help you code in peace:
1. Visual Studio code ( get from moses )
2. Google chrome ( already installed on your computer )
  • Step 2
You are then going to write your first html page!
1. make a folder called 'first project' on your desktop
2. create a new text file and rename it to index.
3. enable the 'show file extension' option on file explorer
4. The file extension in the part after the dot in the file of the name.
5. go to the index file that you created and rename it's file extension to '.html'
6. It should have a microsoft edge sign on it.
7. open visual studio code.
8. follow instructions from moses
9. press CTRL + S on you keyboard to save changes
10. Go to the browser (google chrome) and press F5 on your keyboard to refresh it.
